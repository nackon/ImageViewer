# 画像ビューア 仕様書

## プロジェクト概要
IrfanViewのMac版に相当する、軽量で高速な画像ビューアアプリケーション

## 目的
- フォルダ内の画像を連続して快適に閲覧できる
- キーボード操作で素早く画像を切り替えられる
- ズーム機能で画像の詳細を確認できる
- サムネイル一覧で画像全体を俯瞰できる

## 技術スタック
- **言語**: Rust
- **GUIフレームワーク**: iced (v0.12+)
- **画像処理**: image crate
- **対象OS**: macOS (将来的にはクロスプラットフォーム対応も可能)

## 機能要件

### 1. 画像表示機能
- **対応フォーマット**:
  - JPEG/JPG
  - PNG
  - GIF (静止画)
  - BMP
  - WebP
  - TIFF
  
- **表示モード**:
  - フィット表示（ウィンドウに収まるように表示）
  - 実サイズ表示（100%）
  - 自由ズーム

### 2. キーボードナビゲーション機能

| キー | 動作 |
|------|------|
| `→` / `Space` / `N` | 次の画像 |
| `←` / `Backspace` / `P` | 前の画像 |
| `Home` | 最初の画像 |
| `End` | 最後の画像 |
| `T` | サムネイル一覧表示切り替え |
| `Q` / `Esc` | アプリ終了 |

### 3. ズーム機能

| キー | 動作 |
|------|------|
| `+` / `=` | ズームイン（25%刻み） |
| `-` | ズームアウト（25%刻み） |
| `0` | 実サイズ（100%） |
| `F` | フィット表示 |
| マウスホイール | ズームイン/アウト |

### 4. サムネイル一覧表示機能
- **表示形式**:
  - グリッド表示（4〜6列）
  - サムネイルサイズ: 150x150px（アスペクト比維持）
  - 現在選択中の画像をハイライト表示
  
- **操作**:
  - クリックで画像を選択 → 詳細表示に切り替え
  - `↑` `↓` `←` `→` キーでサムネイル間を移動
  - `Enter` で選択した画像を詳細表示
  - `T` または `Esc` でサムネイル表示を閉じる
  - スクロール可能
  
- **パフォーマンス**:
  - サムネイルは非同期生成
  - 生成したサムネイルはキャッシュ
  - 表示領域外のサムネイルは遅延ロード

### 5. ファイル操作機能
- **フォルダから開く**: 画像ファイルを開くと、同じフォルダ内の全画像を自動で読み込み
- **ソート順**: ファイル名の昇順
- **対応する起動方法**:
  - ファイルをダブルクリック
  - コマンドライン引数でファイルパスを指定
  - ドラッグ&ドロップ（将来対応）

## 非機能要件

### パフォーマンス
- 画像切り替えは200ms以内に完了
- メモリ使用量: 現在表示中の画像 + 前後1枚をプリロード + サムネイルキャッシュ
- 大きな画像（10MB+）でもスムーズに表示
- サムネイル生成は非同期で、UIをブロックしない

### ユーザビリティ
- ウィンドウサイズは前回終了時の状態を記憶
- 最後に開いたフォルダを記憶
- 画面中央に画像を配置
- 画像情報の表示（ファイル名、解像度、ファイルサイズ）

### 保守性
- モジュール構成を明確に
- テストコードを書く
- エラーハンドリングを適切に行う

### カラースキーム
- 背景色: ダークグレー (#2b2b2b)
- テキスト色: ライトグレー (#cccccc)
- アクセント色: ブルー (#4a9eff)
- 選択中サムネイル: ブルー枠 (#4a9eff, 3px)
- サムネイル背景: ミドルグレー (#3a3a3a)
